Los dos tipos de OKR válidos en Neto

Un buen OKR en una empresa operativa puede ser de mejora o de iniciativa nueva — ambos son válidos y necesarios.

Tipo 1 — OKR de mejora operativa

Mejora una métrica que ya existe. El KPI ya se mide permanentemente; el Key Result lo mueve durante el ciclo.

🎯 Objective
Ser la tienda más rentable de la región este trimestre
📏 Key Results
Ticket promedio: $150 → $180
Merma: 3.2% → 2.5%
Disponibilidad anaquel: actual → ≥92%
mueve KPIs existentes

Tipo 2 — OKR de iniciativa nueva

Crea algo que no existía antes. El Key Result mide una métrica nueva surgida de una apuesta estratégica.

🎯 Objective
Construir la primera base de clientes leales de Neto
📏 Key Results
5,000 clientes registrados en programa de lealtad
NPS del programa ≥ 40 al cierre del trimestre
30% de clientes realiza una segunda compra
crea métricas nuevas

💡 Regla práctica para Neto

En cada ciclo define al menos un OKR de cada tipo. El Tipo 1 te mantiene operando bien. El Tipo 2 te hace evolucionar. Una empresa que solo hace Tipo 1 opera mejor pero no cambia. Una que solo hace Tipo 2 innova pero pierde el piso operativo.

✅ Un buen Key Result...

  • Tiene baseline y meta con brecha real
  • Mide un resultado, no una actividad
  • No empieza con "crear / implementar / lanzar"
  • Es ambicioso pero alcanzable en un trimestre
  • Tiene un dueño único responsable

❌ Errores que causan rechazo

  • Poner números en el Objective
  • KR que empieza con verbo de actividad
  • KR sin baseline definido
  • Meta igual o casi igual al baseline
  • Tareas disfrazadas de Key Results

💡 La pregunta que lo decide todo

¿Este número existiría aunque no hubiera OKR este ciclo? Si la respuesta es sí → es un KPI. Si es un número nuevo porque te propusiste cambiar algo → es un Key Result. El KR es el vehículo. El KPI es el destino.
